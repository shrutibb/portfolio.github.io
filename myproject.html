PG Life

About the project

The PG life is a simple website which is made by using all the frontend technologies i.e. HTML, CSS, JavaScript, Bootstrap, React and Backend technologies i.e. DBMS, PHP. The basic goal is to find the best suited PG for the students living at a particular area so that its easier for them to decide which PG to choose for their betterment. It has several webpages including all the details of the PG along with the facilities provided there. All the webpages are highly interactive because of the use of web technologies.

Index of page

First, when we enter the website , the index page or the front page of the website will be shown to the user, from where they can select the city where they want to look for the PG. There are 4 major cities in this project – Delhi, Mumbai, Bengaluru, Hydrabad.

List of properties

After selecting a particular city by the user, the list of all the properties will be shown to them along with their address. There is a filter from where the user can apply filters for the gender for which they want to look up for the PG. The properties can also be sorted from highest to lowest rent and lowest to highest rent.

Viewing the property

After clicking on “view” option, all the details of the PG will be shown along with the ratings and reviews by other customers. Further more details can be seen by the user such as amenities provided by that PG, what other people say about it, ratings etc.

Booking by signing in

If the user wants to book a PG then they have to click on “Book Now” option. After clicking on book now , a box will appear which will ask user to either Login if they already have an account otherwise Signup.
